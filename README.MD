# Pair Play User Engagement Services

This is the backend service that manages user feed, user feed updates, image uploads.


## Requirements 
1. Access to the original AWS account
2. AWS CLI installed
3. Elastic Beanstalk CLI (EB CLI) installed
4. Node.js v22+ & Yarn

## To get started 
1. Clone repo
2. yarn install
3. eb init
4. eb deploy
